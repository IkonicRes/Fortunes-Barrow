Game Instructions

Welcome to Fortune's Barrow!

Instructions for play:

Use the Arrow keys to move your hero

Use space to interact with doors and other objects

The combat is turn-based and attacking will always attack the closest enemy. Combat will give you XP which will determine your score at the end.

Use the action image to cycle through actions, the pass button to skip turn, the run button to run away from combat, and the action button to perform the selected action!

Good luck and have fun!
